# Jarvis_2.0

Jarvis_2.0 is my jarvis IA server.

### Tech
- [Node.js 7.0.0](https://nodejs.org/)

### Installation

```
cd <project root path>
git clone https://github.com/simflouzz59/Jarvis_2.0
cd Jarvis_2.0
node server.js
```
